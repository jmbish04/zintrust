# Broadcasting

Zintrust includes a broadcasting toolkit for publishing events via pluggable drivers.

## Where to look in the codebase

- Toolkit entrypoint: `src/tools/broadcast/Broadcast.ts`
- Drivers: `src/tools/broadcast/drivers/`

## What this page covers

This page documents the framework’s broadcasting primitives at a high level and links you to the core implementation.

If you’re looking for configuration, search the repo for “broadcast” in `src/config/`.
