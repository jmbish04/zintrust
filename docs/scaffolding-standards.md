# Scaffolding Standards

The Zintrust CLI generates consistent project structure so teams can move fast.

## Conventions

- Controllers: `app/Controllers/*Controller.ts`
- Middleware: `app/Middleware/*Middleware.ts`
- Routes: `routes/*.ts` (registered from `routes/api.ts`)
- Core framework code: `src/*`

## Generators

Common generators include:

- `zin add controller <Name>`
- `zin add middleware <Name>`
- `zin add model <Name>`

See the CLI docs for the full list.
