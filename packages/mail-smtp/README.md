# @zintrust/mail-smtp

SMTP mail driver registration for ZinTrust.

- Docs: https://zintrust.com/mail

## Install

```bash
npm i @zintrust/mail-smtp
```

## Usage

```ts
import '@zintrust/mail-smtp/register';
```

Then set `MAIL_DRIVER=smtp` and configure SMTP settings.
