# @zintrust/storage-s3

AWS S3 storage driver registration for ZinTrust.

- Docs: https://zintrust.com/storage

## Install

```bash
npm i @zintrust/storage-s3
```

## Usage

```ts
import '@zintrust/storage-s3/register';
```

Then set your storage disk driver to `s3` and configure AWS credentials.
