# @zintrust/mail-sendgrid

SendGrid mail driver registration for Zintrust.

- Docs: https://zintrust.com/mail

## Install

```bash
npm i @zintrust/mail-sendgrid
```

## Usage

```ts
import '@zintrust/mail-sendgrid/register';
```

Then set `MAIL_DRIVER=sendgrid` and configure `SENDGRID_API_KEY`.
